This is a Full Stack Application created using Node.js, React.js, Express.js and MongoDB.
In order to run ther application you need to have MongoDB and Node.js installed.
Cd into the Server folder, run "npm install" to install all needed dependencies and then run "node app.js" to start the server.
Once the server has started cd into the Client folder and run "npm install" to install all needed dependencies on the client side.
Then run "npm start" on the Client folder and the application should be running on "http://localhost:3000". (Your default browser should pop up, if not just visit "http://localhost:3000" manually)
